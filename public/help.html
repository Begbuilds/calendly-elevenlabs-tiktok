Project Overview

What is the Calendly-Elevenlabs Integration?

This project is a complete server solution that connects Calendly's scheduling API with Elevenlabs' voice agent platform, allowing AI voice agents to handle appointment scheduling over the phone.

The system enables callers to:

  • Talk to an AI voice agent about scheduling
  • Check real-time availability on your Calendly calendar
  • Select a meeting type and preferred time slot
  • Receive a booking confirmation link via SMS
  • Complete the booking process with minimal friction

Best of all, the entire solution requires zero coding knowledge to set up and deploy!

System Architecture

The system connects multiple services to create a seamless scheduling experience:

  ┌───────────────┐     ┌───────────────┐     ┌───────────────┐
  │   Elevenlabs  │     │ Your Backend  │     │    Calendly   │
  │  Voice Agent  │◄────┤    Server     │────►│     API       │
  └───────┬───────┘     └───────┬───────┘     └───────────────┘
          │                     │
          │                     │
          ▼                     ▼
  ┌───────────────┐     ┌───────────────┐
  │    Caller     │     │    Twilio     │
  │  (Phone Call) │     │  SMS Service  │
  └───────────────┘     └───────────────┘

How it works:

  1. Caller phones your Elevenlabs voice agent number
  2. The voice agent checks availability through your backend server
  3. Your server queries the Calendly API for real-time availability
  4. After selecting a time slot, the server sends a confirmation SMS via Twilio
  5. The caller clicks the link to finalize the booking in Calendly

Key Features

Automatic Time Zone Detection

Detects the caller's time zone from their phone number for accurate scheduling.

Smart Availability Checking

Two-stage process: first weekly view, then specific time slots.

Natural Language Conversations

Intuitive dialogue patterns with summarized availability.

Personalization

Dynamically includes current date/time and caller details.

SMS Confirmation

Automatically sends booking links via text message.

Easy Customization

Simple tools for configuring your agent's behavior.
