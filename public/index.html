Getting Started

This server integrates Calendly's scheduling API with Elevenlabs' voice agent platform, allowing voice agents to schedule appointments via phone conversations.

To get started:

  1. Create an agent using the Agent Builder
  2. Customize your agent's prompt with the Prompt Builder
  3. Test your agent through Elevenlabs
